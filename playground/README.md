# TokenKit Playground

This is a playground for testing the `astro-tokenkit` plugin in a real Astro environment.

## Getting Started

From the project root:

```bash
npm run playground
```

Or from this directory:

```bash
npm install
npm run dev
```

## Features Tested

- [x] Integration setup in `astro.config.mjs`
- [x] Automatic middleware injection
- [x] Client-side script injection (`client-init`)
- [x] Idle session monitoring
- [x] Token rotation and session management
